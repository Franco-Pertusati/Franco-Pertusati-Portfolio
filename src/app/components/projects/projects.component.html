Proyectos

I developed a software to efficiently manage table orders, track customer bills, generate and print order slips for both the customer and the kitchen, as well as keep a record of the cash register and work shifts. Additionally, the system stores sales data for future analysis. This project came about while I was working at a bar during my studies. Handling these tasks manually with paper and pencil proved to be inefficient and prone to errors. I wanted to create a digital solution to streamline these processes, so I developed it using my knowledge of HTML, CSS, and JavaScript at the time.
